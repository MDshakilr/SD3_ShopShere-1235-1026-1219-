Description:
 
ShopSphere is a full-featured e-commerce web application built using Laravel, designed to deliver a complete and modern online shopping experience.
The system includes secure user registration and login, efficient product management,
a smooth cart and checkout process with payment gateway integration, and a powerful admin dashboard for system control and monitoring.
The application supports two primary user roles: Customer and Admin, ensuring proper role-based access and functionality.

Project Stack:
1.Backend: Laravel (PHP)
2.Frontend: Blade + HTML + CSS + Tailwind + JavaScript
3.Database: MySQL


Key Features:
1.User Registration and Login
2.Product Catalog with Advanced Filters
3.Cart and Checkout with Payment Gateway Integration
4.Order History Management
5.Chatbot Interaction for Customer Support
6.Auto Search and Autocomplete Functionality
7.Admin Dashboard for Managing Products, Orders, Users, and Analytics
